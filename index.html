Features

Our system here at Arda’s Legends uses builds on the server we call Claimbuilds. These buildings are built by players to claim regions from the Claiming Map. Each build follows a certain set of guidelines that the Admin team will use to check off and accept the build. The guidelines are unique to each build based on the reward you want. The accepted claimbuilds can become siege-able and if all of a faction’s claimbuilds are lost in a region the faction will lose said region. So, happy building!
