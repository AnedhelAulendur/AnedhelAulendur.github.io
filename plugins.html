Custom Plugins and Features


Arda's Plugin

Arda's plugin is a plugin made by Aned. This plugin contains hunt system and build mode.
For now, you can do /hunt and /aid someone in order to hunt/help someone in a hunt event.

*Note that the plugin is still work in progress.


DeathPlugin

The DeathPlugin is a plugin made by Jukoz. This plugin create a corpse on a player's death by placing a skull on a fence, that can be interacted/open upon a rightClick
Players can so die & lose their loots and hunts people to obtain their loot, by respecting hunt rules of course.
Also, once the skull is empty of loot, it container is destroyed and become a simply vanilla skull

*DeathPlugin is working on a HashMap system, it's currently work in progress to save dead people's loot in a case of a server crash, since this event will destroy the hashmap and so the loots of every corpses.


Chair Plugin

The plugin is made by Axel_HiddenIsMe_Rooftop (not us)
Therefore, we took materials id's of of Lord of the Ring mod, and included it in the plugin's config.yml file, so now we can use modded stairs blocks as chairs!

*This plugin teleports the player, so /back will result in your last seat location.

Essentials Economy

We utilize the Essentials Economy feature on the server. Credits for this go to the Contributors to essentials.
A Lord of the Rings Mod coin is having the value of $1 in the economy system. There is a bank in Esgaroth where you can exchange your money for balance and vise versa.
